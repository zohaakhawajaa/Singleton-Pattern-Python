# Python Singleton

A clean **Singleton design pattern** implementation in Python.  
Ensures a class has **only one instance** and provides a **global access point**.  
Useful for configuration management, logging, database connections, or any scenario where a single instance is required.

---

## Features

- Enforces a **single instance** of a class
- Provides a **global access point**
- Easy to integrate in Python projects
